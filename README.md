# Student Form App

A simple web application built with **Node.js** and **Express** that allows users to submit student data via a form. The app collects first and last names and displays a confirmation message upon submission.

## Features

- User-friendly HTML form
- Collects first and last name
- Displays a success message after submission
- Built with Express and Node.js
